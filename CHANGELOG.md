# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-07-04

### Added

- Project foundation and repository structure
- Frontend scaffold with React, Vite, TypeScript, Tailwind CSS
- Backend scaffold with Express, TypeScript, Prisma
- Docker Compose configuration for local development
- CI/CD workflows for GitHub Actions
- Documentation framework
- Shared types and utilities module
- Database design with SQL Server 2022
- Cloudinary integration configuration
